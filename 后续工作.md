# 后续工作

请阅读当前项目及文档后，按下面的要求修改本项目。

1. 调用外部工具，优化本项目的语音播报功能。现在的项目在播报面试问题时，声音听上去很机械，停顿不自然。请你查找，是否存在开源或免费的外部工具，如可以找到，请优化本项目的语音播报功能。
2. 调用外部工具，优化本项目的面试回答功能。在用户回答问题时，构建一个“文字-时间”序列，记录用户回答每个字的时间，然后清洗一下，把语气词都过滤掉，然后统计用户首个核心词汇回答的时间、中间卡壳的时间等数据，并把这个数据也纳入最后的面试评价中（也就是面试评价不仅仅评价用户回答问题的正确与否，还要考虑用户掌握知识点的熟练程度）。你可以参考这个链接https://chatgpt.com/share/6a18242c-c040-83ea-80c8-1dc3a983c702中的内容。
3. 在“知识库管理”页面的下面，基于用户已上传的文档，总结整理归纳出一些“常见考点”，以小卡片的形式展示出一些常见的问题（如“Java中ConcurrentHashMap怎么实现的？”“Redis底层的数据结构有哪些？“等），点击每个卡片后（有翻转动画），可以查看这个问题的答案。
4. 在界面右上角“退出”按钮的旁边，增加一个“日间/夜间模式切换”按钮，可以切换界面的显示风格（浅色/深色）。
